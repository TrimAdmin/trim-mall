# 项目简介

## 定位

Trim Mall是一款多端多功能的商城系统，使用当前最新的技术开发，采用Monorepo仓库管理模式，依赖管理更轻松。

仓库统一使用eslint、prettier、stylelint进行代码规范约束，保证代码风格统一。提交时使用commitlint和husky进行校验，确保提交信息的规范性。

全模块使用Typescript。

## 模块

Trim Mall共包含六个模块。

| 模块名                              | 描述         |
| ----------------------------------- | ------------ |
| [trim-mall-admin](#trim-mall-admin) | 后台管理系统 |
| [trim-mall-app](#trim-mall-app)     | 商城APP/H5   |
| trim-mall-mpwx                      | 微信小程序   |
| trim-mall-doc                       | 项目文档     |
| trim-mall-server                    | 后台服务     |

### trim-mall-admin

后台管理系统在[Fantastic-admin](https://fantastic-admin.gitee.io/)
的基础上二次开发，如果你感兴趣可以去官网看看，是一款非常优秀的后台管理系统模板。

主要技术包括`Vite5`、`Vue3`、`Element Plus`、`Axios`、`scss`、`pinia`等。

| 主模块 | 子模块                                                                                    | 主模块介绍                               |
| ------ | ----------------------------------------------------------------------------------------- | ---------------------------------------- |
| 基础   | 物品（物品、SKU、物品属性、单位）、库存（仓库、货架）、策略（退换货理由、邮费、价格说明） | 对基础信息进行管理                       |
| 仓库   | 出入库（发货、退换货）、库存管理                                                          | 对出入库等流程进行操作                   |
| 电商   | 售前售后服务、用户评价、客服、营销活动（优惠券、商品打折）、会员                          | 电商流程的操作模块                       |
| 报表   | 库存数据、电商数据、用户数据统计                                                          | 数据报表查看模块                         |
| 系统   | 系统管理（用户、角色、部门、数据字典）、全局设置、用户使用意见反馈                        | 对后台管理系统和整个电商系统进行全局设置 |
